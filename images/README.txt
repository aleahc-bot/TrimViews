FONTS  (assets/fonts/)
------------------------------------------------------------------------------
Add your licensed Garet files so headings render in true Garet:
  assets/fonts/Garet-Bold.woff2   (+ .woff / .otf / .ttf optional)
  assets/fonts/Garet-Book.woff2   (+ .woff / .otf / .ttf optional)
Until present, headings fall back to a clean system sans-serif.
